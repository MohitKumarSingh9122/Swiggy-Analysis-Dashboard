# Swiggy Sales Analysis Dashboard
# Project Objective 
This project analyzes Swiggy sales data using Microsoft Excel, applying data cleaning and visualization techniques with charts and pivot tables. It highlights key metrics like total sales, average rating, total orders, and shows sales trends along with state and city performance.

## Dataset used
- [Dataset](https://raw.githubusercontent.com/MohitKumarSingh9122/Swiggy-Analysis-Dashboard/main/Swiggy%20Raw%20Data%20Excel.xlsx)

## Questions (KPIs)
- What is the total sales generated?  
- What is the average order value (AOV)?  
- What is the average customer rating?  
- How many total orders were placed?  
- What is the monthly sales trend?  
- Which day has the highest sales (daily trend)?  
- How do sales vary on a weekly basis?  
- Which food category (Veg vs Non-Veg) generates more revenue?  
- Which states contribute the most to total sales?  
- What are the top 5 cities by sales?  
- How do sales, ratings, and orders compare across quarters?  
- Which restaurants perform best in terms of sales?
- - Dashboard Interaction <a href="https://github.com/MohitKumarSingh9122/Swiggy-Analysis-Dashboard/blob/main/Swiggy%20Analysis%20Dashboard.png">View Dashboard</a>

##  Process
- Data Collection: Imported Swiggy raw dataset into Microsoft Excel  
- Data Cleaning: Removed null values, duplicates, and corrected data types  
- Data Transformation: Organized and structured data using Excel tools  
- Data Analysis: Used Pivot Tables to analyze sales, orders, and ratings  
- Data Visualization: Created charts (bar, line, pie) for trends and insights  
- Dashboard Creation: Built an interactive dashboard with slicers (Month, State, Restaurant)  
- Insights Generation: Identified key trends like top cities, peak sales time, and category performance

## Project Insight 
- Sales show clear peaks during certain months and weekends  
- Veg category generates slightly higher revenue than Non-Veg  
- Top cities contribute a major share of total sales  
- Customer ratings remain consistently high (~4.3)  
- Average order value indicates stable customer spending behavior  
- Certain states dominate overall sales performance

## Conclusion 
The Swiggy Sales Analysis Dashboard provides clear insights into sales trends, customer behavior, and regional performance. It helps in making data-driven decisions by identifying key patterns in orders, ratings, and revenue.
